Module 05

Visual-First Communication

The Format Hierarchy for Busy Humans

Track: EditorialDuration: 60 minFormat: Lecture + Workshop

Overview

Core Thesis

Reading is slow. Your principal is context-switching all day. Default to visual formats. If someone can't absorb your output mid-workday, the output failed — not the person.

"Visual-first, always. Status briefings should be 5 slides you flip through in 90 seconds, not paragraphs."

Key Concepts

What You'll Learn

1. The Format Hierarchy

Slide deck (flip in 90s) → Annotated images → Clickable HTML → Audio summary → Long-form text (last resort).

2. Markdown Is for Agents

"Anything YC is supposed to read must be HTML or a proper document format. Markdown is for agents and systems."

3. Audio Is the Review Layer

"YC cannot sit at a computer to read articles. Convert EVERYTHING that needs review to audio. This directly unblocks the review bottleneck."

4. Dashboard as Dispatch Center

"The dashboard itself should contain the dispatch prompts needed to spin off new agents, eliminating the need to go anywhere else."
