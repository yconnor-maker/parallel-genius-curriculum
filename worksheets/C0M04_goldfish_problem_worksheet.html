The Goldfish Problem

Module C0-04
Time: 30 minutes
Format: Individual + practical
Prerequisite: C0-03
Part I

The Amnesia Audit

Think about the last 5 AI sessions you had. For each one, answer:

How much time did you spend re-explaining context that a previous session already knew?
Were there any decisions from a past session that you forgot and had to re-make?
How many of your AI sessions start with "Here's what we discussed last time..." or similar re-briefing?
Part II

Build Your Context File

⏱ 15 Minutes

Exercise: Create PROJECT_CONTEXT.md

For your primary project, write a markdown file that an AI can read to instantly understand your work. Include:

Project Name & One-Line Description:
Who am I? (Role, expertise, what makes your perspective unique)
Key decisions already made (things that are LOCKED and shouldn't be revisited):
Brand voice / tone rules (how should AI-generated text sound?):
Things to NEVER do (mistakes, anti-patterns, forbidden approaches):
Current status (what's done, what's in progress, what's blocked):
Part III

The Cold Start Test

⏱ 5 Minutes

Exercise: Before vs. After

Open a fresh AI chat. Ask a project-specific question WITHOUT your context file. Then start a new chat, paste your context file, and ask the same question.

The question you asked:
Quality of the answer WITHOUT the context file (1-10):
Quality of the answer WITH the context file (1-10):
What was different? What did the context file enable?

"The first infrastructure you build is not an AI agent. It's a system that remembers what you decided."

— From C0-M04
Part IV

Persistence Stack Assessment

Rate your current setup for each layer of the persistence stack:

Layer 1 — Session Persistence (auto-save, localStorage): Do you have this? Y/N. Describe:
Layer 2 — Project Context Files: Do you have one? Is it current? When was it last updated?
Layer 3 — Structured Memory: Do you have a decision registry, knowledge base, or conversation logs?
Layer 4 — Cross-Agent State: If you use multiple AI tools, do they share context? How?
Completion

Before Moving to C0-05

  • I completed my Amnesia Audit and know how much time I waste re-explaining
  • I created a PROJECT_CONTEXT.md for my primary project
  • I completed the Cold Start Test and measured the difference
  • I can name the 4 layers of the Persistence Stack
  • I understand that persistence is the FIRST infrastructure to build
