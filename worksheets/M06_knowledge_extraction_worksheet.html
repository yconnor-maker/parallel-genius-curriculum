Key Concepts

ConceptDefinition
Zero Persistent MemoryAI retains nothing between sessions. Not a bug — a design constraint to build around.
Extract → Route → DeleteGet knowledge out, put it somewhere permanent, THEN clean up.
Extraction Memo5 sentences at session end: worked on, decided, key output, still open, context for next self.
