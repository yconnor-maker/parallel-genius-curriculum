Reference

Key Concepts

ConceptDefinition
Format HierarchySlide deck > annotated images > clickable HTML > audio > long-form text. Always start at the top.
Markdown = Agent FormatAgents read markdown. Humans get HTML, slides, or audio. Never deliver markdown to humans.
Audio Review LayerConvert review items to audio so busy stakeholders can review while mobile.
Dashboard = DispatchThe dashboard isn't passive display — it's an active dispatch center with trigger buttons.
