The Program

What This Curriculum Is

Most AI courses teach you to write prompts. This one teaches you to run an AI-augmented operation — the difference between asking a chatbot a question and managing a team of autonomous agents across multiple businesses, databases, and creative projects simultaneously.

Built from lived experience running 8+ AI agents across clinical medicine, fiction publishing, editorial operations, and organizational infrastructure — every lesson comes from a real session, a real failure, or a real breakthrough.

⚡

Not Replacement — Infrastructure

AI compresses the cycle time between vision and execution. The talent, the vision, the lived experience — that's you. AI handles the logistics of talent.

✏️

The Editor, Not the Author

AI is like the editor you want — the one who works for you. Strong on feedback, strong on line editing, not trying to impose their vision on yours.

🔥

From the Hardest Places

Some of the best creative work comes from the hardest periods. AI catches the fragments in real time — in the car, at 2 AM, mid-crisis — and builds from there.

Capstone

The Content Moat

AI capabilities are easily replicated. Proprietary clinical experience is not. The true business moat is not the agent architecture — it is the translation of direct bedside practice into unique, proprietary atomic units.

One 5-minute bedside dictation correctly placed in the pipeline spawns an EMR note, a 6-episode teaching series, 9 distinct clinical IP tokens, and newsletter distributions. You don't need more ideas. You need magnifiers for the work already done.

What We've Learned

9 Patterns From the Field

01

Governance-Enforcement Gap

Excellent rules, poor enforcement. Writing a rule ≠ building the infrastructure to comply.

02

Verification Failures Compound

Unchecked claims become embedded in downstream work. Every agent output must be independently verified.

03

Founder as Architect

The founder's value is direction, not production. AI agents that present choices get answered. Lists get ignored.

04

Voice as Highest-Value Input

Dictation is the highest-leverage activity. 10x faster than typing, and you don't self-censor.

05

System Entropy Accelerates

Multi-agent systems generate entropy faster than any person can organize. Periodic consolidation is mandatory.

06

The Re-Discovery Tax

Agents re-discover facts that were already known because findings live only in narrative signouts.

07

Security Debt Is Silent

Credentials in public repos. Nobody notices until someone looks. AI optimizes for "make it work."

08

Circuit Breakers Save Everything

Agents that know when to stop are more valuable than agents that always finish.

09

The Format Is the Velocity

Structured cards with recommendations = resolved. Unstructured narrative = stalled decisions.

The Evidence

Lesson Library

Every lesson comes from a real session, a real failure, or a real breakthrough. Not theory — lived operational experience running AI agents at scale.

Real Failure

The 196-Article Hallucination

An agent falsely attributed 196 articles. Roadmaps were built on this fabrication. Not discovered for weeks.

Real Failure

If Decisions Don't Persist, They Were Never Made

Review dashboards with comment boxes had no save functionality — decisions evaporated on page close.

Real Failure

The PAT That Shipped to GitHub

An Airtable token shipped publicly in source code. Anyone who saw it could read/write the entire database.

Real Failure

The Phantom File Problem

4 sessions across 2 agents claimed files that don't exist at stated paths. A signout saying "Created File X" is a claim, not a fact.

Real Failure

When Agents Delete

Two Rule X violations in one week: 6 files deleted from one repo, 5+ from another. Deletion rules must be absolute.

Workflow

Two Agents, One Table, Opposite Directions

697 records to triage. One agent on A-end, one on Z-end. Impossible to collide. Zero-infrastructure concurrency.

Workflow

The Dictation-to-Novel Pipeline

60 minutes of dictation → 52K-word Book 1, 9 new scenes, serialization guide, epub, trilogy outline.

Workflow

AI Reading Handwritten Clinical Notes

The Doctor agent read 38 pages of scanned clinical notes and matched patients across a 110-row encounter log.

Workflow

Dispatch Agent Multi-Project State

One person, multiple businesses, multiple AI agents. A dispatch agent consolidates state to a single dashboard.

Decision

"Stop Being Lazy" — The Founder Feedback Loop

AI agents default to one-shot delivery. Founders need choices, not deliverables. Architecture → Templates → Content.

Decision

12 Decisions as Card Deck

Instead of a wall of text, decisions delivered as structured HTML cards with accept/override options.

Decision

The Hitchhiking Problem

Different agents have different capacities. If you get a slow one, say thank you, get out, and hitchhike again.

Infrastructure

Schema Drift Between Code and Database

Humans rename fields in the UI. Agents read field names from code. Drift is inevitable. Build for rename tolerance.

Infrastructure

The Sentinel's Stop Condition

44 potential fixes found, threshold was 25. It stopped. The right response to unexpected scale is to stop and report.

Infrastructure

62% Prompt Compliance Failure

69 of 111 prompts don't reference the governance document. A constitution only works if wired into every execution point.

Infrastructure

The Reconciliation Tax

40-50% of time spent chasing agent outputs. The agents are smart enough. The problem is they can't see each other.
